改造要点

A

单个大窗口承载所有组件

不再一个组件一个 Window，而是用一个覆盖桌面的全屏窗口装所有组件。拖动时只改 Canvas 坐标，不触发 OS 窗口移动，解决卡顿。退出编辑后只显示组件区域，其余透明穿透。

B

手机桌面尺寸系统

小：迷你条 80×32 / 小方 80×80 / 小圆 80×80；中：横条 240×120 / 竖条 120×240 / 中方 160×160；大：大方 240×240 / 大竖 160×320 / 大横 320×160。每个组件提供 2-4 种尺寸。

C

ClassIsland 2.1 式全屏编辑

进入编辑后全屏半透明遮罩，显示网格线，组件可拖拽、缩放、删除。底部固定工具栏：重置视图、外观、添加组件、完成。右键组件可选尺寸/删除。

D

首次运行强制欢迎页

检测是否首次运行 → 弹出欢迎向导 → 完成后才创建组件。如果配置文件存在但没有 IsFirstRun 标记，也算首次运行。
