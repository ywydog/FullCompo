# FullCompo 欢迎向导设计

## 1. 问题描述

当前首次启动弹出的 `WelcomeWindow` 内容过于简单，只有一个标题、几句说明和一个按钮，用户反馈“窗口啥也没有”。需要参考 ClassIsland 的欢迎向导，提供结构化的首次设置流程。

## 2. 目标

- 首次启动时展示一个多步骤欢迎向导
- 引导用户完成外观主题、快捷设置等基本配置
- 向导完成后立即在桌面显示配置好的面板
- 界面风格与 ClassIsland 类似：左侧步骤条 + 右侧内容区 + 底部导航按钮

## 3. 设计方案

### 3.1 向导结构

4 步向导：

1. **欢迎页**
   - 软件图标/Logo
   - 大标题：欢迎使用 FullCompo
   - 一句话介绍：在桌面上搭建你的个性化信息面板
   - 「下一步」按钮

2. **外观主题**
   - 标题：选择外观
   - 主题选项：浅色 / 深色 / 跟随系统
   - 小预览区域展示主题色样
   - 「上一步」「下一步」按钮

3. **快捷设置**
   - 标题：快捷设置
   - 复选框：开机自动启动
   - 复选框：创建桌面快捷方式（Windows 平台）
   - 「上一步」「下一步」按钮

4. **完成页**
   - 标题：开始使用
   - 操作提示列表：
     - 右键点击面板进入编辑模式
     - 编辑模式下可拖动、添加、删除组件
     - 右键系统托盘图标打开设置或退出
   - 「完成」按钮，关闭向导

### 3.2 布局

```
+------------------------------------------+
|  [Logo]  FullCompo                       |
|                                          |
|  [Step 1]  欢迎                          |
|  [Step 2]  外观                          |
|  [Step 3]  快捷设置                      |
|  [Step 4]  完成                          |
|                                          |
+--------------------------------+---------+
|                                |         |
|        内容区                   |  预览区  |
|                                | (可选)   |
|                                |         |
+--------------------------------+---------+
|                    [上一步] [下一步]     |
+------------------------------------------+
```

实际实现采用 Avalonia `Grid`：
- 左侧 `ListBox` 作为步骤导航，禁用直接点击（按顺序前进）
- 右侧 `ContentControl` 根据当前步骤切换视图
- 底部 `StackPanel` 放置导航按钮

### 3.3 窗口属性

- `Width = 720`, `Height = 480`
- `WindowStartupLocation = CenterScreen`
- `CanResize = false`
- 标题：`欢迎使用 FullCompo`

### 3.4 数据与配置

向导结束时统一保存：
- `AppSettings.ThemeId`：根据选择的主题更新
- `AppSettings.RunOnStartup`：根据复选框更新
- `AppSettings.IsFirstRun = false`
- 调用 `_configService.Save()`

桌面快捷方式通过 Windows COM/Shell 实现，失败时不阻塞。

### 3.5 主题预览

在外观页，提供一个小的可视化预览：
- 一个带圆角的矩形
- 背景色使用当前主题的 `BackgroundColor`
- 文字颜色使用 `ForegroundColor`
- 实时跟随 `ThemeBox` 选择变化

### 3.6 错误处理

- 单页视图异常不阻塞向导，记录日志并显示占位文本
- 保存配置失败时弹出错误提示，但仍关闭向导
- 创建桌面快捷方式失败时静默忽略

## 4. 验收标准

- [ ] 首次启动显示 4 步欢迎向导
- [ ] 左侧步骤条正确显示当前步骤
- [ ] 外观页可切换主题并实时预览
- [ ] 快捷设置页可配置开机自启和桌面快捷方式
- [ ] 完成页显示基本操作提示
- [ ] 点击完成后向导关闭，桌面面板使用新配置
- [ ] 非首次启动不再显示向导
